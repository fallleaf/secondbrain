### 🚀 核心功能

1. **完整 1-9 优先级系统**
   - 中央政府文件 (9级) → 权重 2.0
   - 部委文件 (7级) → 权重 1.6
   - 公司文档 (5级) → 权重 1.2
   - 个人笔记 (3级) → 权重 1.0
   - 网络收集 (1级) → 权重 0.8

2. **混合检索算法**
   - 关键词搜索 (SQLite FTS5)
   - 语义搜索 (FAISS 向量相似度)
   - RRF 融合 (Reciprocal Rank Fusion)
   - 优先级加权

3. **MCP 工具集**
   - `semantic_search` - 混合搜索
   - `read_note` - 读取笔记
   - `list_notes` - 列出笔记
   - `write_note` - 写入笔记
   - `delete_note` - 删除笔记
   - `move_note` - 移动笔记
   - `search_notes` - 关键词搜索
   - `get_index_stats` - 索引统计
   - `rebuild_semantic_index` - 重建索引
   - `get_priority_config` - 优先级配置
