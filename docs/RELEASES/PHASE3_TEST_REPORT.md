## 测试场景

```python
# 编码 50 个文档
embeddings = embedder.encode(texts, batch_size=256)
```

```python
# 构建 100 文档索引
for doc in documents:
    embedding = embedder.encode_single(doc["content"])
    index.add_document(doc["id"], embedding, metadata)
```

## 生产环境建议

### 1. 索引规模

| 文档数 | 构建时间 | 搜索延迟 | 内存占用 |
|--------|----------|----------|----------|
| 100 | 5.5s | 0.65ms | ~5MB |
| 1,000 | ~55s | ~1ms | ~50MB |
| 10,000 | ~9min | ~2ms | ~500MB |
| 100,000 | ~1.5h | ~5ms | ~5GB |

### 2. 批量处理策略

- **小批量 (<100)**: 直接处理
- **中批量 (100-1000)**: 分批处理，每批 256 条
- **大批量 (>1000)**: 异步队列 + 定期索引重建

### 3. 索引维护

- **增量更新**: 实时添加新文档
- **定期重建**: 每天/每周全量重建
- **版本管理**: 索引文件带时间戳
