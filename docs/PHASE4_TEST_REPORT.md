## 🎯 功能亮点

### 1. 多维度统计

- **基本统计**: 文档数、分块数、标签数、链接数
- **存储统计**: 数据库大小、各表记录数
- **分布统计**: 文档类型分布、优先级分布

### 2. 智能重建

- **全量重建**: 清空并重新索引所有文件
- **增量更新**: 只处理变更的文件
- **进度回调**: 实时显示重建进度

### 3. 性能优化

- **VACUUM**: 回收空闲空间
- **REINDEX**: 重建所有索引
- **ANALYZE**: 更新统计信息
