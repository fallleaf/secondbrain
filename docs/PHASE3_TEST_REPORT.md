## 🎯 功能亮点

### 1. 智能链接提取

支持多种链接格式：
- Markdown: `[text](link)`
- Obsidian: `[[link]]`
- 带锚点：`[[link#anchor]]`
- 带显示文本：`[[link|display]]`

### 2. 断裂链接检测

自动识别断裂链接：
- 目标文档不存在
- 目标文件路径错误
- 非文档链接（人名、日期等）

### 3. 孤立笔记分析

识别未被链接的笔记：
- 帮助发现"知识孤岛"
- 建议建立关联

```

```

**核心功能**:
- `scan_all_links()` - 扫描所有文件链接
- `find_broken_links()` - 查找断裂链接

- `get_links()` - 获取出站链接
- `get_note_tags()` - 获取笔记标签
- `add_tag()` / `remove_tag()` - 标签管理
